Process

How it works

01

Create or Join a Project

Sign up and explore available projects or start your own project. Connect with students who want to collaborate.

02

Get Assigned Tasks

Project owners create and assign tasks with defined difficulty levels and deadlines.

03

Complete Tasks & Submit Work

Work on the task using external tools and submit evidence such as pull requests, commits, or documentation links.

04

Earn XP & Build Reputation

Climb the rankings, earn mentor badges, and build a verified portfolio of contributions.

Gamification

XP & Reputation System

Every contribution counts. Earn ratings, build reputation, and climb the rankings.

XP System Details

  • Easy Task 10
  • Medium Task 20
  • Hard Task 40
  • Early Completion Bonus 5

Ratings are awarded across effort. Task appraisal is used to increase level in the system.

Reputation

  • Task Approved +20
  • Mentor Badge +20
  • Revision Required -5
  • Missed Deadline -5
  • Leaving Project -20

30 Days inactive = Reputation × 0.95 decay
